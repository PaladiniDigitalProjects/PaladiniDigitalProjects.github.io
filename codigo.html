¿Qué es codificar?

Dentro del entorno web, la codificación se entiende como el proceso de convertir una interficie visual en código fuente para poder integrarla dentro de un sistema informático.

Actualmente, una web se codifica mediante los lenguages de etiquetado HTML, CSS y el lenguaje orientado a objetos Javascript.

Este proceso normalmente lo realiza el frontend developer.

HTML


Hypertext Markup Language

Lenguaje basado en etiquetas que se abren y cierran. La etiqueta contiene información y al mismo tiempo asigna valor al contenido.


<html></html>
<header></header>
<body></body>
<header></header>
<h1></h1>
<p></p>

CSS


Cascading Style Sheet

Lenguaje basado atributos que se asocian a una etiqueta HTML mediante una clase o identificador.

La posición de la clase en la hoja de estilos delimita su giraría en base al resto de definiciones.


.titular {
font-family: Helvetica;
color: red;
font-size: 34px;
}



Servicios online como JSFidle o Codepen, nos ofrecen herramientas para poder trabajar en pequeñas piezas de código, permitiendo compartirlas con la comunidad online.




Frontend developer

Término que aparece en contraposición al backend developer, perfil encargado de la indexación del contenido en la base de datos mediante un gestor de contenidos (CMS).

El frontend developer se encarga de la codificación de la parte frontal en la que interactúan los usuarios, mediante la interficie gráfica.


La figura del frontend developer ha de tener un perfil mixto.

Ha de comprender la lógica de la programación , y tener una sensibilidad estética adecuada para modular, sistematizar, y codificar correctamente la interficie gráfica.
