El sitema del sistema

Normas y valores

Este sistema ha sido pensado para que sea funcional y neutro para que destaquen al máximo las figuras y imágenes.

A nivel de tecnología se a trabajado con JADE ,para el HTML, y conSass para los estilos CSS.
La rejilla es el de Bootstrap,el resto esta realizado desde cero.
Como repositorio de versiones se ha utilizado Github.
