Librería de componentes

Una librería de componentes funciona como un repositorio de todos los elementos diseñados, y muestra una vista global del sistema.

Resulta imprescindible para compartir el trabajo realizado y modular futuros componentes.

Comparte y documenta el sistema, incluso define los estilos CSS, para que tu equipo este sincronizado y que todos puedan aportar al sistema.
