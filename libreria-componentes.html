Libreria de componentes

Una librería de componentes funciona como un repositorio de todos los elementos diseñados, y muestra una vista global del sitema.

Resulta imprescidible para compartir el trabajo ya realizado, y modular componentes.

Documenta y comparte.

Comparte y documenta, incluso define estilos CSS, para que tu equipo este sincronizado y que todos puedan aportat al sistema.
