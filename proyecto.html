El proyecto

Esta guía nace con la voluntad de ser un repositorio abierto donde unificar ideas, definiciones y buenas prácticas para la creación de sistemas editoriales para web. Esta espacio se plantea como una primera versión y existe la voluntat de poder ser ampliada con más secciones.

Para cualquier errata, sugerencia o mejora me pueden contactar en daniel@paladini.cat

Este proyecto se enmarca dentro de un proceso de investigación académica a través del diseño, dentro del Máster de Diseño y Comunicación (MUDiC) de la escuela universitaria
