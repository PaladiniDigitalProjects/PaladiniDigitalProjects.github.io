¿Que es codificar?

Dentro del mundo web, la codificación se entiende como el proceso de convertir una interficie visual en código fuente, que permita integrar la interficie dentro de un sistema informático.

Actualmente, una web se codifica mediate los lenguages de etiquetado HTML, CSS y el lenguaje orientado a objetos Javascript.

HTML


Hypertext Markup Language

Lenguage basado en etiquetas que se abren y cierran, donde el continente delimita y asigna valor al contenido.


<html></html>
<header></header>
<body></body>
<header></header>
<h1></h1>
<p></p>

CSS


Cascading Style Sheet

Lenguage basado atributos que se asocian a una etiqueta HTML mediante una clase o id.


.titular {
font-family: Arial;
color: red;
font-size: 34px;
}



Servicios conline como JSFidle o Codepen, nos ofrecen heramientas para poder coficar pequeñas piezas de código compartiéndolo con la comunidad online.
